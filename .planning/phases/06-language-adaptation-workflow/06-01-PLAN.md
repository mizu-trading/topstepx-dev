<objective>
Create the adapt-language workflow that converts existing TopStepX trading bots between supported languages (JavaScript/TypeScript and Python), using inline language profiles for extensibility and mandatory safety verification.

Purpose: Users need to convert working trading bots between languages while preserving all safety guardrails (SAF-01 through SAF-05). Without this workflow, language conversion is manual and error-prone -- safety patterns get dropped, library mappings are guessed, and the trading build order is ignored. This workflow makes conversion systematic, safe, and repeatable.

Output: New file `topstepx/workflows/adapt-language.md` (~600-900 lines) following the established TSX workflow XML tag structure with 8 numbered steps, inline language profiles, and a mandatory safety verification gate.
</objective>

   - Validate source != target (error if same)

   - Include any unrecognized libraries flagged in Step 2
